Parking Pro PWA

Cómo usar:
1. Sube estos archivos a un hosting HTTPS como Netlify, Vercel o GitHub Pages.
2. Abre la web desde Chrome en Android.
3. Menú de Chrome → Agregar a pantalla principal.

Importante:
- La cámara para leer QR normalmente requiere HTTPS.
- Si abres el HTML localmente, la app funciona, pero la cámara puede no funcionar.
- Los datos se guardan en el navegador/dispositivo.
